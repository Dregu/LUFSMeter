Installation
------------

It's not allowed to redistribute the VST SDK. Therefore you have to
get it from Steinberg (vst_sdk2_4_rev2.zip) and copy the ``pluginterfaces``
and ``public.sdk`` folders into the ``vstsdk2.4`` directory.
